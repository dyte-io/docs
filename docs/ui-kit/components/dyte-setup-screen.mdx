# dyte-setup-screen

A screen shown before joining the meeting, where you can edit your display name,
and media settings.

```html
<dyte-setup-screen id="dyte-el" style="height: 360px"></dyte-setup-screen>

<script>
  const el = document.getElementById('dyte-el');
  el.meeting = meeting;
</script>
```
