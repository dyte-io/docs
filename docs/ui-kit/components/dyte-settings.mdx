# dyte-settings

A settings component to see and change your audio/video devices
as well as see your connection quality.

```html
Desktop View:
<dyte-settings class="dyte-el"></dyte-settings>

Mobile View:
<dyte-settings size="sm" class="dyte-el"></dyte-settings>

<script>
  const elements = document.getElementsByClassName('dyte-el');
  for (const el of elements) {
    el.meeting = meeting;
  }
</script>

<style>

    height: 480px;

  }
  dyte-settings[size='sm'] {
    max-width: 360px;
  }
</style>
```
