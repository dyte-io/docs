# dyte-settings

A settings component to see and change your audio/video devices as well as see
your connection quality.

```html
Desktop View:

Mobile View:
<dyte-settings size="sm" class="dyte-el"></dyte-settings>

<script>
  const elements = document.getElementsByClassName('dyte-el');
```
