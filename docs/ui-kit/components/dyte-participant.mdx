# dyte-participant

A participant entry component used inside `dyte-participants` which shows data
like: name, picture and media device status.

You can perform privileged actions on the participant too.

```html
<dyte-participant id="dyte-el"></dyte-participant>

<script>
  const el = document.getElementById('dyte-el');
  el.participant = participant;
</script>
```
