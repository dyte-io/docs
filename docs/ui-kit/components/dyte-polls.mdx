# dyte-polls

A component which lists all available plugins a user can access with the ability
to enable or disable them as per their permissions.

```html
<dyte-polls id="dyte-el"></dyte-polls>

<script>
  document.getElementById('dyte-el').meeting = meeting;
</script>

<style>
  dyte-polls {
    height: 480px;
    width: 100%;
    max-width: 320px;
    background-color: '#000';
  }
</style>
```
