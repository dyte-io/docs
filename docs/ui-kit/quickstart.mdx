# Quickstart

Quickly get started with UI Kit, with just a few lines of code!

First you would need to setup your project with the provided hooks and the
`<DyteProvider />`.

```html
<body>
  <dyte-meeting id="my-meeting"></dyte-meeting>

  <script>
    const init = async () => {
      const meeting = await DyteClient.init({
        authToken: '',
        roomName: '',
        defaults: {
          audio: true,
          video: true,
        },
      });

      document.getElementById('my-meeting').meeting = meeting;
    };

    init();
  </script>
</body>
```

:::tip

Try adding `show-setup-screen="true"` prop on `<dyte-meeting />` to enable the
Setup Screen.

:::
