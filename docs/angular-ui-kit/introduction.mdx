# Introduction

:::info

Our UI Kit documentation is under development. Please check back here soon for
the updated version!

:::

A set of UI components to truly customize your Meeting UIs, in Angular.

:::info

The following packages are available for different frameworks:

- [React UI Kit](/react-ui-kit/).
- [Angular UI Kit](/angular-ui-kit/)
- Vue UI Kit (docs coming soon!)

:::

## Features

<ul className="emoji-list">
  <li>
    <span>🌐</span> Built with Web Components.
  </li>
  <li>
    <span>🚀</span> Blazingly fast. Uses browser provided ShadowDOM.
  </li>
  <li>
    <span>🤏</span> Small bundle size (&lt;500KB) for the entire UI (less if you
    use individual components).
  </li>
  <li>
    <span>🏎</span> Transfers only the code you need over the network as
    components are lazy loaded.
  </li>
  <li>
    <span>✨</span> ShadowDOM provides style encapsulation, your CSS styles
    wouldn't affect our styles and vice versa.
  </li>
</ul>

Read this [blog post](https://dyte.io/blog/custom-ui-kit-sdk/) to know more
about the UI Kit project.
