# Quickstart

Get the `roomName` for the meeting and the participant's `authToken` from our
backend APIs (read more about our server APIs [here](/api/)), and pass them to
Dyte's client SDK. To launch the meeting in your app the setup is
straightforward. This will launch a fullscreen view with your meeting.

<TabItem value="kotlin">

```kotlin
import dyte.io.uikit.DyteUIKitBuilder
import dyte.io.uikit.DyteUIKitConfig
import io.dyte.core.models.DyteMeetingInfo

// Configure meeting parameters
val meetingInfo = DyteMeetingInfo(
    orgId = "YOUR_ORG_ID_HERE",
    roomName = "ROOM_NAME_HERE",
    authToken = "AUTH_TOKEN_HERE"
)
// pass in activity context for the SDK
val config = DyteUIKitConfig(
    activity = this,
    dyteMeetingInfo = meetingInfo,
)
// initialise an instance
val dyteUIKit = DyteUIKitBuilder.build(config)
// launch the activity
dyteUIKit.loadUi()
```

<TabItem value="java">

```java
import dyte.io.uikit.DyteUIKit;
import dyte.io.uikit.DyteUIKitBuilder;
import dyte.io.uikit.DyteUIKitConfig;
import io.dyte.core.models.DyteMeetingInfo;

// Configure meeting parameters
DyteMeetingInfo meetingInfo = new DyteMeetingInfo(
    "YOUR_ORG_ID_HERE",
    "ROOM_NAME_HERE",
    "AUTH_TOKEN_HERE",
    true, // enable audio
    true, // enable video
    "BASE_URL_HERE",
    "this is name" // display name here
);
// pass in activity context for the SDK
DyteUIKitConfig config = new DyteUIKitConfig(
    this,
    meetingInfo
);
// initialise an instance
DyteUIKit dyteUIKit = DyteUIKitBuilder.INSTANCE.build(config);
// launch the activity
dyteUIKit.loadUi();
```
