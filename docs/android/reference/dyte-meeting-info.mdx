# DyteMeetingInfo

| Property name | Data type | Required                | Use                                                 |
| ------------- | --------- | ----------------------- | --------------------------------------------------- |
| authToken     | string    | Yes                     | Auth token for the particpant                       |
| roomName      | string    | Yes                     | Meeting name to join                                |
| audioEnabled  | boolean   | No (defaults to `TRUE`) | weather or not enabled their audio while joining in |
| videoEnabled  | boolean   | No (defaults to `TRUE`) | weather or not enabled their video while joining in |
| orgId         | string    | Yes                     | Org id, available at dev portal                     |
