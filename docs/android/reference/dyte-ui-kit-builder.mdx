# Dyte UI Kit

## DyteUIKitBuilder

DyteUIKitBuilder is the main entry point for your app to talk to the SDK. It
mainly exposes two objects. First is `DyteMobileClient` which helps attach
listeners for all the events which are supported in our core SDK. Second object
is DyteUIKit which helps you navigate between UI elements which are wrapped
inside UI Kit SDK.

## DyteUIKitConfig

DyteUIKitConfig is an builder class for Dyte UI Kit that is used to create
instance of Dyte UI Kit client.
