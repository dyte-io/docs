# Installation

1. Install the client SDK from npm

```bash npm2yarn
npm install @dytesdk/mobile
```

2. Install external dependencies required by our SDK

```bash npm2yarn
npm install react-native-webview react-native-webrtc @react-native-async-storage/async-storage react-native-vector-icons
```

3. Add this to the root of your project

```js
import { registerGlobals } from 'react-native-webrtc';

registerGlobals();
```

## Android

:::info

The below instructions are for the release builds, debug builds should work
without any additional steps.

:::

1. Edit your `android/gradle.properties` and add the following line

```
android.enableDexingArtifactTransform.desugaring=false
```

2. Create / append to the file `android/app/proguard-rules.pro`

```
-keep class org.webrtc.** { *; }
-dontwarn org.chromium.build.BuildHooksAndroid
```

3. In your `android/app/build.gradle` edit the release configuration and add the
   following line importing the proguard configuration

```
buildTypes {
  release {
    ...
    ...
    ...
    proguardFiles getDefaultProguardFile('proguard-android.txt'), 'proguard-rules.pro'
  }
}
```

:::info

If you encounter chinese characters in your app instead of icons, please apply
this fix mentioned in
[react-native-vector-icons issue](https://github.com/oblador/react-native-vector-icons/issues/1106#issuecomment-574683931)

:::

## iOS

:::tip iOS minimum version 12.0

We support minimum OS version `12.0` for our iOS SDK.

:::

1. Open your podfile and make sure your platform is set to ios 12

```ruby
platform :ios, '12.0'
```

2. Add the fonts and permission entries in `info.plist`

```xml
<key>UIAppFonts</key>
<array>
    <string>MaterialCommunityIcons.ttf</string>
    <string>MaterialIcons.ttf</string>
</array>
<key>NSBluetoothPeripheralUsageDescription</key>
<string>We will use your Bluetooth to access your Bluetooth headphones.</string>
<key>NSBluetoothAlwaysUsageDescription</key>
<string>We will use your Bluetooth to access your Bluetooth headphones.</string>
<key>NSCameraUsageDescription</key>
<string>For people to see you during meetings, we need access to your camera.</string>
<key>NSMicrophoneUsageDescription</key>
<string>For people to hear you during meetings, we need access to your microphone.</string>
<key>NSPhotoLibraryUsageDescription</key>
<string>For people to share, we need access to your photos.</string>
<key>UIViewControllerBasedStatusBarAppearance</key>
<false/>
```
