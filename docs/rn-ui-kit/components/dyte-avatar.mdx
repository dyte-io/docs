# DyteAvatar

Avatar component which renders a participant's image or their initials.

```jsx
<Row>
  <DyteAvatar size="sm" />
  <DyteAvatar size="md" participant={meeting.self} />
  <DyteAvatar size="lg" />
</Row>
```
