# DyteChatToggle

A button which toggles visibility of chat.

You need to pass the `meeting` object to it to see the unread messages count
badge.

When clicked it emits a `dyteStateUpdate` event with the data:

```ts
{
  activeSidebar: boolean;
  sidebar: 'chat' | 'none';
}
```

```jsx
<Center>
  <DyteChatToggle size="sm" meeting={meeting} />
  <DyteChatToggle size="lg" meeting={meeting} />
  <DyteChatToggle variant="horizontal" size="sm" meeting={meeting} />
</Center>
```
