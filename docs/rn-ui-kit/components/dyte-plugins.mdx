# DytePlugins

A component which lists all available plugins from their preset, and ability to
enable or disable plugins.

```jsx
<Center>
  <DytePlugins
    meeting={meeting}
    style={{ height: '480px', maxWidth: '320px', backgroundColor: '#000' }}
  />
</Center>
```
