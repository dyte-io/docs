# DyteMenu

A menu component.

```jsx
<View style={{ padding: '52px 0 0 0' }}>
  <DyteMenu placement="top">
    <DyteButton>Top Menu</DyteButton>
    <DyteMenuList>
      <DyteMenuItem onClick={() => alert('You clicked: alert()')}>
        alert()
      </DyteMenuItem>
    </DyteMenuList>
  </DyteMenu>

  {/* This menu will be placed at top due to less space */}
  <DyteMenu placement="bottom">
    <DyteButton>Bottom Menu</DyteButton>
    <DyteMenuList>
      <DyteMenuItem onClick={() => alert('You clicked: alert()')}>
        alert()
      </DyteMenuItem>
    </DyteMenuList>
  </DyteMenu>
</View>
```
