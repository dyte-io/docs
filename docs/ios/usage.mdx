# Basic Usage

## Configure the meeting connection

You need to pass a [`DyteMeetingConfig`](./reference/connection-config) to
[`DyteMeetingView`](./reference/dyte-meeting) to define which meeting should the
client connect to and a couple of other setup parameters, out of which only
`authToken`, `roomName`, `width` and `height` are mandatory.

<Tabs
groupId="ios-language"
defaultValue="swift"
values={[
{ label: 'Swift', value: 'swift', },
{ label: 'Obj-C', value: 'obj-c', },
]
}>
<TabItem value="swift">

```swift
import DyteSdk;

let  config = DyteMeetingConfig();
meetingConfig.roomName = "YOUR_ROOM_NAME";
meetingConfig.authToken = "YOUR_AUTH_TOKEN";
```

</TabItem>
<TabItem value="obj-c">

```objectivec
@import DyteSdk;

DyteMeetingConfig *meetingConfig = [[DyteMeetingConfig alloc] init];
meetingConfig.roomName = @"YOUR_ROOM_NAME";
meetingConfig.authToken = @"YOUR_AUTH_TOKEN";
```

</TabItem>
</Tabs>

## Choose between programmatic initialization of view or view initialization using Storyboard

### Programatic view initialization

On a button click or similar user action, you can call the following code.

<Tabs
groupId="ios-language"
defaultValue="swift"
values={[
{ label: 'Swift', value: 'swift', },
{ label: 'Obj-C', value: 'obj-c', },
]
}>
<TabItem value="swift">

```swift
let dyteView = DyteMeetingView(frame: CGRect(x: 0, y: 0, width: self.view.bounds.size.width, height:self.view.bounds.size.height ))
self.view.addSubview(dyteView)
```

</TabItem>
<TabItem value="obj-c">

```objectivec
DyteMeetingView *dyteView = [[DyteMeetingView alloc] initWithFrame:CGRectMake(0.0, 0.0, meetingConfig.width, meetingConfig.height)];
[self.view addSubview:dyteView];
```

</TabItem>
</Tabs>

### Storyboard view initialization

1. Add a View in Storyboard and set it's custom class to `DyteMeetingView`.
   Similar to above example, if you want to use `tag`, you can set that too in
   Storyboard.

![xcode-screenshot-showing-storyboard-setup](/static/ios/ios-setup.png)

2. Reference the view in your code and initiate the meeting.

<Tabs
groupId="ios-language"
defaultValue="swift"
values={[
{ label: 'Swift', value: 'swift', },
{ label: 'Obj-C', value: 'obj-c', },
]
}>
<TabItem value="swift">

```swift
let dyteView = self.view as DyteMeetingView
```

</TabItem>
<TabItem value="obj-c">

```objectivec
DyteMeetingView *dyteView = (DyteMeetingView *) self.view;
```

</TabItem>
</Tabs>
