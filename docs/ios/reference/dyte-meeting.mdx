# DyteMeetingView

DyteMeetingView is an iOS UIView that can be included in your iOS application to
show the Dyte Meeting UI. It exposes the usual view properties and lifecycle
methods that are expected from any view.

## Properties

| Property name | Data type                                             | Use                                                                                       |
| ------------- | ----------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| delegate      | id<[DyteMeetingViewDelegate](./dyte-meeting-events)\> | Pointer which refers to the object that implements the `DyteMeetingViewDelegate` protocol |

## Methods

| Method name    | Params (with data type)                                                               | Use                                                                                                                                                                                        |
| -------------- | ------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| join           | config: [DyteMeetingConfig](./connection-config)                                      | Join a Dyte meeting with given config parameters                                                                                                                                           |
| updateUIConfig | uiConfig: NSDictionary (which is JSON dictionary of [DyteUIConfig](./dyte-ui-config)) | Change the UI and layout of the current meeting according to the relevant params specified in the object (see [customize the meeting UI](./../customize-meeting-ui) for available options) |
