Plugins allow you to host interactive, collaborative, and effective meetings.
You can use plugins like Remote Browser, YouTube, Whiteboard, Excalidraw, and many more to collaborate when having a Dyte meeting.

Plugins SDK allows developers to build, test and deploy Plugins.

:::danger Important
Plugin SDK does not support the old Web SDK. Please use the [new Web SDKs](http://localhost:3000/react-ui-kit).
:::
