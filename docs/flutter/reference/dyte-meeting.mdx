# DyteMeeting (Flutter widget)

`DyteMeeting` is a Flutter widget that can be used in your Flutter application
to show the Dyte Meeting UI.

## Properties

| Property name   | Data type                             | Required                 | Use                                                                                                                                                                                                                      |
| --------------- | ------------------------------------- | ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| onInit          | function(meeting: DyteMeetingHandler) | Yes                      | On successful meeting creation, returns the meeting to the callback function provided (see [DyteMeetingHandler object](./meeting) for more details)                                                                      |
| authToken       | string                                | Yes                      | Auth token for the particpant                                                                                                                                                                                            |
| roomName        | string                                | Yes                      | Meeting name to join                                                                                                                                                                                                     |
| showSetupScreen | boolean                               | No (defaults to `false`) | Show audio / video setup screen to the participant                                                                                                                                                                       |
| autoTune        | boolean                               | No (defaults to `true`)  | Dyte uses VP9 codec by default, but switches to H264/VP8 depending on participants video codec support. VP9 is a superior codec but not all browsers support it. If you disable `autoTune` Dyte will use VP8 by default. |
| uiConfig        | [<String, dynamic>](./dyte-ui-config) | No                       | To customize ui meeting                                                                                                                                                                                                  |
